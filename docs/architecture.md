## 화면별 구성

### 랜딩 페이지 (`/`)
- 핵심 문구: "당신의 독서를 하나의 여정으로 보여드립니다"
- 예시 독서 지도 목업 (하드코딩): 데미안 → 싯다르타 → 이방인
- 서비스 3줄 소개 (추천 아닌 해석/연결/시각화)
- Google로 시작하기 버튼
- 푸터: 서비스 소개, 문의

### 책 검색 페이지 (`/books/search`)
- 검색창 (책 제목 / 저자명)
- 검색 결과 카드 (표지, 제목, 저자, 출판사)
- 상태 선택 버튼 (READ / READING / WISHLIST)
- 이미 등록된 책: "등록됨" 표시 + 상태 변경 가능
- 예외: 결과 없음 / 카카오 API 오류 시 Google Books 폴백 / 네트워크 오류 토스트

### 내 서재 페이지 (`/library`)
- 상단: 사용자 이름 + 책 등록 수 요약
- 탭: 읽은 책 / 읽는 중 / 읽고 싶은 책
- 책 카드: 표지, 제목, 저자, 상태 변경, 삭제
- 하단 고정 CTA: **독서 지도 생성하기** 버튼
  - READ 3권 미만: 비활성화 + 툴팁
  - READ 3권 이상: 활성화
- 예외: 서재 비어있음 → 검색 이동 유도

### 독서 지도 결과 페이지 (`/reading-map`)
```
당신의 독서 지도

[독서 유형]        자아 탐구형 독자
[관심 주제]        실존 · 자유 · 정체성 · 고독
[독서 흐름]        데미안 → 싯다르타 → 이방인
[현재 위치]        실존주의 입문 단계
[다음 독서 경로]   시지프 신화 → 존재와 무
[이유 설명]        사용자의 독서 흐름은 자기 발견에서...

[독서 지도 다시 생성하기] [분석 기록 보기]
```
- 예외: 분석 결과 없음 / AI 분석 실패 / 분석 중 로딩

## 사용자 흐름

### 첫 방문
```
랜딩 → Google OAuth → 내 서재(비어있음) → 책 검색 → 등록
→ 독서 지도 생성(READ 3권↑) → AI 분석 로딩 → 결과 확인
```

### 재방문
```
자동 로그인(세션 유지) → 내 서재 → 새 책 등록 or 지도 재생성
→ 최신 독서 지도 확인
```

---

## MVP 범위

**포함**
- Google OAuth 로그인
- 카카오 책 검색 + 등록 + 상태 관리
- 내 서재 (READ / READING / WISHLIST)
- 독서 지도 생성 (AI 분석)
- 독서 지도 결과 화면
- 분석 기록 조회 (최신 1건)

**제외 (MVP 이후)**
- 커뮤니티, 팔로우, 댓글, 좋아요
- 리뷰 작성, 공개 프로필, 별점, 랭킹
- 알림, 결제, 모바일 앱
- 다국어(영어), 소셜 공유
- 알라딘 파트너스 링크 (수익화)
